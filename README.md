> 게시판 제목, 작성자, 조회수, 등록일

> 게시물 
> 게시물 제목, 내용, 작성자, 조회수, 등록일
> 등록: 게시물 등록 페이지로 이동
> 목록: 게시물 목록 페이지로 이동
> 수정: 현재 보고 있는 게시물을 수정(비밀번호 요구)
> 삭제: 현재 보고 있는 게시물을 삭제(비밀번호 요구)
> 댓글 등록: 작성자, 비밀번호, 내용
> 댓글: 작성자, 내용, 삭제, 공감, 대댓글
